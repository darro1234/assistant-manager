Manager+ PWA

Upload all files to the root of your hosting or GitHub Pages repository.

Important:
- Open through https:// or localhost to enable PWA installation and offline cache.
- Opening index.html directly with file:// works as a page preview, but service worker/PWA offline mode will not activate.
- Main files: index.html, manifest.json, service-worker.js, browserconfig.xml, favicon files, Apple touch icon and PWA icons.
